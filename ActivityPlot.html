Analyse de l'activité des paquets et dépôts R¶

Ce notebook a pour but de fournir quelques tapes (et le code associ) permettant de visualiser l'activit des dpts et des paquets R. Dans un premier temps, nous chargeons les noms et propritaires des dpts R qui correspondent un package R (autrement dit : qui ont un fichier DESCRIPTION). Ensuite, nous chargeons la liste des vnments du flux d'activits de chacun de ces dpts, et nous comparons par groupe : les dpts qui ne sont pas des packages, les dpts qui sont des packages, les dpts qui sont des packages SAUF ceux de cran et rpkg (deux owner particuliers).

Chargement du flux d'activités¶

Nous allons extraire les informations du flux d'activits. En particulier, nous garderons : - Le nom et le propritaire du dpt, - Le type de l'vnement, - La date de l'vnement

In [101]: 
import pymongo
import dateutil.parser
from itertools import imap

conn = pymongo.MongoClient()

def _from_json_to_flat(e):
    return {'type': e['type'], 
                   'created_at': dateutil.parser.parse(e['created_at']),
                   'owner': e['repository']['owner'],
                   'repository': e['repository']['name']}

events = imap(_from_json_to_flat, 
              conn.r.events.find({'type': 'PushEvent'},
                                 fields=['type', 'created_at', 'repository.name', 'repository.owner']))
In [102]: 
# This should take some time (expect 683425 items)
df_events = pandas.DataFrame(list(events))

Graphiquement...¶

Le graphique suivant reprend une moyenne glissante sur 30 jours du nombre d'vnements de type "PushEvent". Chaque courbe correspond une partie (ou le tout) des dpts : - la courbe bleue correspond aux dpts contenant un package R li au compte cran, - la courbe verte est invisible, mais elle correspond un tout petit trait aux alentours d'avril 2014 (un peu avant le dbut de la courbe bleue) et correspond aux dpts contenant un package R li rpkg, - la courbe rouge correspond aux dpts contenant un package R, qui n'appartient ni rpkg, ni cran. - la courbe cyan correspond l'ensemble de tous les dpts lis R.

Pour procder cette identification, nous comparons le champ Package des fichiers DESCRIPTION des packages R de Github avec ceux de CRAN.
