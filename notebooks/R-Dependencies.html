Analyse des dépendances des paquets R sur CRAN et GitHub¶

Nous allons analyser les dpendances des diffrents paquets R prsents sur CRAN et sur GitHub, afin de voir s'il est possible d'identifier une diffrence de comportement entre ceux qui sont uniquement dvelopps/publis sur GitHub et ceux qui sont uniquement dvelopps/publis sur CRAN.

Ce notebook ncessite d'avoir accs aux fichiers DESCRIPTION des paquets R, qu'ils soient de CRAN ou de GitHub.

We need to parse Depends and Imports to extract the related packages. We only consider the name of the package (not its version), i.e. a contiguous sequence of characters, so it is easily expressed as follows:

In [6]: 
def parse_packages_list(pkg_list):
    return map(str.strip, map(lambda p: p.split('(')[0], pkg_list.split(',')))

# Replace every NaN in Depends and Imports by an empty string (i.e. an empty list of packages)
pkg = pkg.fillna(value={'Depends': '', 'Imports': ''})

# Count 'Depends' and 'Imports' items
pkg['dependencies'] = pkg['Depends'].map(lambda p: len(parse_packages_list(p))) + pkg['Imports'].map(lambda p: len(parse_packages_list(p)))

# We want a true list of dependencies
def clean_dependencies(row):
    d1 = parse_packages_list(row['Depends']) if len(row['Depends']) > 0 else []
    d2 = parse_packages_list(row['Imports']) if len(row['Imports']) > 0 else []
    return ', '.join(d1 + d2)

def _dependencies_not_in_cran(df, package_name, filter_=False):
    """
    Need on_cran global variable.
    Set filter_ to True if you want only the dependencies that are NOT on cran.
    """
    if filter_:
        filter_ = lambda x: x[1] == False
    else:
        filter_ = lambda x: True
    return (package_name, filter(filter_, 
                            zip(df.loc[package_name]['dep_list'].split(', '), 
                                map(lambda x: x in on_cran, df.loc[package_name]['dep_list'].split(', '))
                            )
                        )
            )
