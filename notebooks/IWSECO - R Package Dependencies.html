def parse_dependencies(item):
    depends = item['Depends'] if item['Depends'] != pandas.np.nan else ''
    imports = item['Imports'] if item['Imports'] != pandas.np.nan else ''
    f = lambda lst: [dep.split('(')[0].strip() for dep in lst.split(',')]
    return filter(lambda x: len(x) > 0, f(depends) + f(imports))

La classe suivante sert de base notre graphe. Les attributs sont :

  • name: nom du package
  • sources: liste des sources o ce paquet peut tre trouv
  • source: source principale (dans l'ordre indiqu)
  • dependencies: liste des dpendances (attendu : instances de Package)

La mthode add_source permet d'ajouter une source (str). La liste dependencies peut tre manipule directement.

In [63]: 
class Package(object):
    sources = ['R', 'cran', 'bioconductor', 'rforge', 'github']
    
    def __init__(self, name):
        self.name = name
        self.sources = []
        self.dependencies = []
        
    def add_source(self, source):
        if source not in self.sources:
            self.sources.append(source)
            
    @property
    def source(self):
        for source in Package.sources:
            if source in self.sources:
                return source
        return 'Unknown'
    
    def __unicode__(self):
        return '{name} on [{sources}]'.format(name=self.name, sources=', '.join(self.sources))
    __str__ = __repr__ = __unicode__

Pour chaque paquet identifi, nous ajoutons ces sources. Le dictionnaire packages vise associer chaque nom de paquet une instance de la classe Package.

Ajoutons les dpendances. Si le paquet n'est pas connu de notre liste de paquets, nous crons tout de mme une instance de Package. La source associe ce paquet sera automatiquement Unknown et ce paquet pourra donc tre utilis de faon transparente lors des calculs sur le graphe.

for name, data in df.iterrows():
    p = packages[name]
    for dep in parse_dependencies(data):
        p_dep = packages.get(dep, Package(dep))
        packages[dep] = p_dep
        p.dependencies.append(p_dep)

Enfin, il nous reste crer le graphe dirig. Ce graphe est construit classiquement pour les dpendances. A noter que les dpendances de la source Unknown sont galement dans ce graphe (qui n'est naturellement pas une composante connexe).

import networkx

dg = networkx.DiGraph()
for package in packages.itervalues():
    dg.add_node(package)
    for dependency in package.dependencies:
        dg.add_edge(package, dependency)

def d2df(gd):
    m = {}
    for package, value in gd.iteritems():
        m[package.name] = {'source': package.source, 'value': value}
    return pandas.DataFrame.from_dict(m, orient='index')

df = d2df(networkx.in_degree_centrality(dg))

for name, group in df.groupby(by='source'):
    fig = plt.figure()
    ax = fig.add_subplot(111)
    group['value'].plot(kind='hist', figsize=(8,4), bins=10, title=name, ax=ax)
