Sur base des donnes extraites dans github-RPackage-repository.csv (Github), dans cran-description.csv (CRAN), bioconductorsvn_description.csv (BioConductor) et r-forge_description.csv (R-Forge), nous allons regarder la rpartition des packages de Github au sein des diffrentes communauts.

Pour chacune de ces sources de donnes, nous allons conserver les dpendances ainsi que les auteurs et ajouter une tiquette pour la "source" de ce package (Github, CRAN, Bioconductor et R-Forge). Si un package est prsent sur plusieurs sources, nous utiliserons les donnes prioritairement dans l'ordre suivant : Cran, Bioconductor, R-Forge et enfin, Github.

Prparation du dataset Github Il n'y a pas grand chose faire sur ce dataset, car il a t prpar de sorte fitter au mieux ce qui allait suivre. Il est, par contre, ncessaire de slectionner les dpts dits "canoniques" (ie. un seul dpt pour chaque valeur de Package) pour respecter les index. Pour rappel, un dpt canonique signifie que le dpt n'est priori pas un fork, et est le plus vieux dpt cr parmi l'ensemble des dpts hbergeant le mme package.

cran_imports = temp.query('key == "Imports"').rename(columns={'value': 'Imports'})[['Package', 'Imports']]
cran_depends = temp.query('key == "Depends"').rename(columns={'value': 'Depends'})[['Package', 'Depends']]
_ = temp[['Package']].drop_duplicates().merge(cran_author, how='left', on='Package')
_ = _.merge(cran_imports, how='left', on='Package')
cran = _.merge(cran_depends, how='left', on='Package')

Nous allons maintenant regrouper les diffrents datasets en un seul, en se basant sur le nom du package. Comme indiqu prcdemment, un ordre va tre appliqu pour rcuprer les informations qui sont prsentes pour un mme package provenant de plusieurs sources (typiquement le cas pour Github).

Nous remplissons les N/A par les valeurs adquates, et nous obtenons un joli petit dataset avec toutes les informations utiles sur la provenance. Il nous reste ajouter une cl "Github only" pour indiquer qu'un package n'est prsent que sur Github.

def _f(row):
    if row['github'] == 1:
        if row['cran'] == row['rforge'] == row['bioconductor'] == 0:
            return 1
        else:
            return 0
    else:
        return 0
    
packages['Github only'] = packages.apply(_f, axis=1)
